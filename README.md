## 게시판 프로젝트

### 프로젝트 개요
- 개발 기간 : `2024.08.26` ~ `2024.09.06`
- 개발 인원 : 1명
- 해당 프로젝트는 커뮤니케이션을 위해 서로의 의견을 공유하고 이야기할 수 있는 웹 사이트입니다.




<br>

### 사용 기술
- FE : JavaScript, React, Axios, TailWind,
- BE : Java 17, SpringBoot 3.3.3, Spring Data JPA, Spring Security, QueryDsl, Gradle, MySQL
